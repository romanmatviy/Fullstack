UI/UX design

I design convenient and aesthetic interfaces in Figma. A good UI/UX means more conversions, less bounces and happy users.

My approach to design

I approach design as a developer — I design interfaces that can really be implemented in code. I follow modern trends: dark mode, glassmorphism, micro-animations. But the main goal is to solve the business problem and make the user happy.

What is included in the design

  • ✓
    Wireframes — skeleton models for approval of the structure.
  • ✓
    UI design in Figma — full design of all screens and states.
  • ✓
    Design system — components, colors, typography.
  • ✓
    Adaptive design — Desktop, Tablet and Mobile layouts.
  • ✓
    Interactive prototypes — clickable prototype for presentation.
  • ✓
    Design + layout — as a designer and developer in one person, I implement design in code.

Frequently asked questions

Will I be able to make changes to the design myself?

Yes, I give full access to the Figma files. You will be able to edit texts and images yourself.

How many revisions are included?

2-3 rounds of edits are included in the base price. Each revision is discussed before starting.

Do you also design?

Yes, I am both a designer and a developer. I can both create a layout in Figma and implement it in code.

Do you design for mobile apps?

Yes, I design interfaces for web applications and responsive versions for mobile devices.
