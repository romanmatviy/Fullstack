UI/UX Design

Проектую зручні та естетичні інтерфейси в Figma. Добрий UI/UX — це збільшення конверсії, зменшення відмов та задоволені користувачі.

Мій підхід до дизайну

Я підходжу до дизайну як розробник — проектую інтерфейси, які реально можна якісно реалізувати в коді. Стежу за сучасними трендами: dark mode, glassmorphism, micro-animations. Але головна мета — вирішити задачу бізнесу та зробити користувача щасливим.

Що входить у дизайн

  • ✓
    Wireframes — скелетні макети для затвердження структури.
  • ✓
    UI дизайн у Figma — повноцінний дизайн всіх екранів і станів.
  • ✓
    Design system — компоненти, кольори, типографіка.
  • ✓
    Адаптивний дизайн — Desktop, Tablet та Mobile макети.
  • ✓
    Інтерактивні прототипи — клікабельний прототип для презентації.
  • ✓
    Дизайн + верстка — як дизайнер і розробник в одній особі, я реалізую дизайн у коді.

Frequently Asked Questions

Will I be able to make design changes myself?

Yes, I transfer full access to Figma files. You will be able to edit texts and images yourself.

How many revisions are included?

2–3 rounds of revisions are included in the base cost. Each revision is discussed beforehand.

Do you also code the design?

Yes, I am both a designer and a developer. I can create a mockup in Figma and implement it in code.

Do you design for mobile apps?

Yes, I design interfaces for web applications and responsive versions for mobile devices.
